# Simple FPS Game

一个简单的 Python FPS 射击游戏，使用 Ursina 引擎开发。

## 操作说明

| 按键 | 功能 |
|------|------|
| W/A/S/D | 移动 |
| 鼠标 | 视角 |
| 左键 | 射击 |
| 空格 | 跳跃 |

## 下载

在右侧 "Releases" 或 "Actions" 中下载 Windows 版本。

## 本地运行

```bash
pip install ursina
python game.py
```

## 打包

推送代码到 GitHub 后，会自动构建 Windows EXE 版本。
