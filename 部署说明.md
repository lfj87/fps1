# 🚀 部署说明

## 注册 GitHub 账号

1. 访问 https://github.com/signup
2. 免费注册账号
3. 创建新仓库（名字随意，比如 `fps-game`）

## 推送代码

```bash
cd /mnt/mjxz/ai/game/fps1

# 配置 Git（第一次需要）
git config --global user.name "你的用户名"
git config --global user.email "你的邮箱"

# 关联 GitHub 仓库
git remote add origin https://github.com/你的用户名/fps-game.git

# 提交并推送
git commit -m "Initial commit"
git push -u origin master
```

## 自动打包

推送后：
1. 访问 https://github.com/你的用户名/fps-game/actions
2. 等待构建完成（约 3-5 分钟）
3. 在 "Artifacts" 里下载 `SimpleFPS.exe`

## 本地运行（可选）

```bash
pip install ursina
python game.py
```
